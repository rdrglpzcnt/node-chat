# node-chat 

## Description
Super simple chat made with Express.js.  
You can see who is typing.  
You have to regiter and login.  

## Requirements
Node.js 11.* or later

## Technologies
Node.js with Express.js for backend.  
Socket.IO for real time communication.  
MongoDB for user storage only.  

## Installation
``` bash
npm install
npm run start
```
